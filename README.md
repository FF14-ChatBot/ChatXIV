# ChatXIV

A chat-first assistant that provides extensive, accurate data to *Final Fantasy XIV* (FFXIV) players—from BiS and relic progress to rotations, unlocks, patch history, and raid plans.

## Documentation

- **[Business Requirements Document (BRD)](docs/design-documents/BRD.md)** — Product scope, requirements, and success criteria.
- **[git-crypt setup](docs/git-crypt-setup.md)** — How encryption for `docs/design-documents/` and `docs/tasks/` works, one-time setup, and triage/fix steps for new owners.
- **Developer setup:** [docs/README.md](docs/README.md)

## Overview

ChatXIV reduces the manual labor of researching and retrieving FFXIV data outside the game by answering common questions in one place, with sourced and up-to-date information.
